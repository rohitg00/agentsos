# Identity Directory

This directory contains identity template files for the AgentSOS system.

## File Index

| File | Purpose |
|------|---------|
| SOUL.mdx | Agent personality, values, boundaries, and voice template |
| USER.mdx | User information, preferences, and context template |
| MEMORY.mdx | Persistent memory architecture, schema, and eviction policy |
| AGENTS.md | Multi-agent coordination rules, delegation, and shared state |
| TOOLS.mdx | Available tools reference with permissions and namespaces |
| BOOTSTRAP.mdx | First-run initialization steps and troubleshooting |
