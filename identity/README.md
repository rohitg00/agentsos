# Identity Files

This directory contains template files that define agent personality, user context, memory format, coordination rules, tool references, and initialization instructions.

## Files

| File | Purpose |
|------|---------|
| SOUL.md | Agent personality and values template |
| USER.md | User information template |
| MEMORY.md | Persistent memory format |
| AGENTS.md | Multi-agent coordination rules |
| TOOLS.md | Available tools reference |
| BOOTSTRAP.md | First-run initialization instructions |
